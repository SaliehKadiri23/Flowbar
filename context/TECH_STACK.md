This project will be built using the following technologies. Do not deviate from this stack.

- **Platform:** Google Chrome Extension
- **Manifest Version:** Manifest V3
- **Core Language:** JavaScript (ES6+)
- **Popup/Options UI:** HTML5, CSS3. No heavy frameworks like React or Vue for the UI components. Keep it lightweight and vanilla.
- **Background Logic:** Service Worker (`background.js`)
- **Key APIs:** `chrome.storage.sync`, `chrome.alarms`, `chrome.scripting`, `chrome.tabs`, `chrome.windows`, `chrome.runtime`
- **Code Style:** Follow the Prettier code style for formatting. Use clear, well-commented code.
